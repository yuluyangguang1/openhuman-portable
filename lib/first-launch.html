快速开始

1启动 OpenHuman
双击启动器后，OpenHuman 会自动启动。首次运行会显示Welcome欢迎页面。
所有数据存储在便携包data/目录内，不会污染宿主机。
2选择运行模式
欢迎页面会询问登录方式。选择Continue Locally (实验性)可跳过登录直接使用本地模式。
也可通过 Google / GitHub / X / Discord OAuth 登录（需要网络）。
3密钥环授权
首次启动桌面版会弹出Keyring Consent对话框，询问是否允许本地加密存储 API Key。
点击Accept授权（数据加密存储在便携包内）。

1打开设置

2进入 AI & Models

3启用供应商
面板顶部有一排供应商芯片（OpenAI、Anthropic、DeepSeek 等）。
点击要启用的供应商芯片 → 弹出对话框 → 粘贴 API Key → 确认。
启用后芯片变为高亮状态。
4添加自定义供应商
如果供应商不在列表中，点击Add Custom Provider按钮。
填写：名称、OpenAI 兼容的 API URL、API Key。
支持任何 OpenAI 格式的 API 端点。

常见问题

数据存储在哪里？
所有数据存储在便携包data/.openhuman/目录内。通过OPENHUMAN_WORKSPACE和HOME环境变量重定向，不会在宿主机留下任何痕迹。
可以使用多个供应商吗？
可以。在 AI & Models 面板中启用多个供应商，OpenHuman 会根据任务类型自动选择最佳模型（Managed 模式），或你可以手动指定每个任务使用的供应商（Advanced 模式）。
OpenRouter 是什么？
OpenRouter 是一个统一 API 网关，一个 Key 可以访问 OpenAI、Anthropic、Google 等几乎所有模型。适合不想注册多个供应商的用户。
如何更新 OpenHuman？
下载新版便携包，将data/目录复制到新版即可保留所有配置和数据。data/.home/目录也需一并复制。
U 盘拔出后数据还在吗？
所有数据都在 U 盘的data/目录内。拔出 U 盘后，宿主机上不会留下任何数据。
