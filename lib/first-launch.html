配置 API Key

1 打开设置
OpenHuman 启动后，点击左下角 ⚙️ 设置 图标，或按 ⌘ + , 打开设置面板。
2 添加供应商
在设置中找到 Inference 或 模型配置 部分。
点击 Add Provider，选择你的 AI 供应商，填入 API Key。
3 选择模型并保存
选择一个模型（如 GPT-4o、Claude Sonnet 等），点击保存。
回到主界面即可开始对话。
